한국 전통 관상학 통설 지식 DB 구축을 위한 학술·검증 가능한 출처 조사를 의뢰합니다.

[배경 컨텍스트]

얼굴 사진을 입력으로 받아 한국 운세·인상 분석을 제공하는 한국어 SaaS 제품의
관상 풀이 모듈을 개발 중입니다. 본 모듈은 두 단계 파이프라인으로 구성됩니다:

- 1단계: Vision LLM이 사진을 보고 순수 해부학적 형태 (이마·눈썹·눈·코·입·턱·뺨·
  광대뼈·기색)를 JSON으로 산출. 학파 용어는 사용 금지.
- 2단계: 자연어 LLM이 (1단계 JSON + MediaPipe 478 키포인트 결정론 점수 +
  본 DB의 학파 통설) 세 가지 입력을 받아 운학 도사 사극풍 자연어로 풀이.

핵심 설계 제약: 2단계 LLM이 자신의 사전학습된 관상 통설을 사용하면 사실성
원칙 위반. 학파 통설은 **반드시 본 시스템 코드 DB 출처에서만** 흘러나와야 함.
따라서 검증된 학술·문화재 출처로 영속화된 통설 DB가 필요.

[조사 목표]

한국 전통 관상학의 **표준 통설 30개 이상**을 다음 5개 카테고리로 분류해
각 항목별로 학파 출처·운명 매핑·해당 해부학 형상을 정리한 구조화 데이터를 도출.

각 항목은 반드시 다음 4가지를 충족해야 함:
1. 출처 검증 가능 (KCI 논문 DBpia URL, 한국민족문화대백과사전 encykorea.aks.ac.kr,
   교보문고·알라딘 ISBN 등 라이브 URL)
2. 학파 명시 (마의상법 / 유장상법 / 달마상법 / 신상전편 / 한국 민간 통설 등)
3. 해부학적 영역·형상 명확 (Vision LLM이 산출한 JSON 필드와 매칭 가능해야 함)
4. 운명·인상 매핑 본문 인용 (원문 그대로 — 가짜 인용 금지)

[카테고리 5종]

카테고리 A. 삼정(三停) — 얼굴 세로 3분할
- 상정(上停): 발제~눈썹 아래 (이마 영역)
- 중정(中停): 눈썹~코끝 (눈썹·눈·코 영역)
- 하정(下停): 코끝~턱 (입·턱 영역)
- 항목별 운명 매핑 (초년·중년·말년 운, 학업·재물·자녀 등)
- 학파별 비율 권장치 (3등분 균형 강조 학파 vs 차등 학파)

카테고리 B. 12궁(十二宮) — 얼굴 영역 12분할
- 명궁(命宮), 관록궁(官祿宮), 재백궁(財帛宮), 전택궁(田宅宮),
  처첩궁(妻妾宮), 자녀궁(子女宮), 복덕궁(福德宮), 형제궁(兄弟宮),
  부모궁(父母宮), 노복궁(奴僕宮), 천이궁(遷移宮), 질액궁(疾厄宮)
- 각 궁의 해부학적 위치 (예: 명궁=미간/印堂)
- 각 궁의 운명 매핑 본문
- 학파별 명칭 변형 (예: 처첩궁 vs 첩궁 vs 어미)

카테고리 C. 코 부위 형상 분류 (재백궁 세분류)
- 복코, 매부리코, 들창코, 마늘코, 사자코, 화살코, 주먹코 등 표준 명칭 풀
- 각 형상의 해부학 정의 (콧대 기울기, 콧방울 크기, 코끝 모양, 콧구멍 노출도)
- 각 형상의 운명/성격 통설 매핑
- 한국 통설 vs 중국 학파 통설 비교

카테고리 D. 부위별 형상 분류 (코 외)
- 눈썹: 초승달(신월미), 일자, 청수, 옅음, 짙음, 끊긴 눈썹 등
- 눈: 단봉안, 삼백안(상·하), 봉안, 용안, 음양안, 도화안 등 (13가지 안상 포함)
- 입: 앙월구, 복주구, 사자구, 잉어구, 일자입 등
- 입술: 두께·색깔별 통설
- 귀: 칼귀, 후귀, 송곳귀, 귓볼 두께별 통설
- 각 항목별 해부학 정의 + 운명/성격 매핑

카테고리 E. 학파 출처 메타데이터
- 마의상법(麻衣相法) — 송대 진단(陳摶) 추정, 한국 보급 시기
- 유장상법(柳莊相法) — 명대 원충철(袁忠徹), 한국 보급
- 달마상법(達磨相法) — 인도 달마대사 전승 추정
- 신상전편(神相全編) — 명대 진단 외 편찬
- 한국 민간 통설 — 민족문화대백과 표제 "관상" 등
- 각 학파의 한국 도입 연혁 + 영향력 범위

[요구사항 — 각 항목별 반드시 답할 것]

1. 항목명 (한글·한자·로마자):
2. 카테고리 (A/B/C/D/E):
3. 해부학적 영역·형상 (Vision JSON 필드와 매칭 가능한 정량 정의):
   - 예: 복코 → "콧방울 두툼함 (nostril_wing.thickness 두툼한),
     코끝 둥글다 (nose.tip 둥근), 콧구멍 노출 낮음"
4. 학파 출처 (마의상법 §장·절 / 신상전편 §권 / 민족문화대백과 표제 등):
5. 통설 본문 인용 (원문 또는 한글 번역 그대로):
6. 운명/인상 매핑 (재물운·관운·시간 운 등):
7. 출처 URL (라이브 검증 가능):
   - DBpia 논문: https://www.dbpia.co.kr/journal/articleDetail?nodeId=...
   - 민족문화대백과: https://encykorea.aks.ac.kr/...
   - 교보문고·알라딘 ISBN 페이지
   - 학술지 PDF 직접 링크
8. 출처 검증 일자 (조사 시점):
9. 학파별 변형 (다른 학파에서 같은 형상을 다르게 해석하는 경우):
10. 한계·반증 (현대 인체측정학 통계와의 충돌 또는 학파 내 이견):

[출력 형식 — YAML 스켈레톤 + 마크다운 표]

각 항목별 YAML 블록:

```yaml
- name_ko: 복코
  name_hanja: 福鼻
  category: C
  anatomy_match:
    nose.nostril_wing: 두툼한
    nose.tip: 둥근
    nose.nostril_visibility: 낮음
  schools:
    - school: 마의상법
      passage: "코가 두툼하고 깨끗하면 평생 돈 걱정 없이 편안하게 산다"
      fate_mapping: 재물운 좋음, 재물 축적·관리 능력 탁월
      source_url: https://...
      verified_date: 2026-MM-DD
    - school: 한국 민간 통설
      passage: "둥글고 풍부하며 콧구멍이 잘 보이지 않는 형태 = 큰 부자 상"
      source_url: https://encykorea.aks.ac.kr/...
      verified_date: 2026-MM-DD
  modern_anthropometry_note: "한국인 인체측정학 송우철 2017 콧방울 평균과 매칭 가능"
  caveats:
    - 학파별 이견 (유장상법은 일부 다른 해석)
    - 현대 관상 통설은 매부리코 부정 인식이 약화됨
```

전체 30개 이상 항목을 위 형식으로 작성. 마지막에 마크다운 요약 표:

| ID | 항목 | 카테고리 | 학파 수 | 출처 URL 수 |
|---|---|---|---|---|

[검증 기준]

- 모든 출처 URL은 조사 시점 라이브 검증 통과해야 함 (HTTP 200 + 본문 인용 매칭)
- "학파 통설" 인용은 원문 또는 공식 한글 번역만 — 의역·재해석 금지
- 한국 한정 통설은 학술지 또는 민족문화대백과 직접 인용 필수
- 가짜 인용 (저자·연도·학술지 검증 실패) 발견 시 해당 항목 제외
- DBpia·민족문화대백과·교보문고·알라딘 URL 우선
- 블로그·SNS 출처는 보조 자료로만 사용 (주요 학파 인용 X)
- 학파 권위 인용 시 한국 도입 연혁 명시 (예: 마의상법 조선 후기 보급)
- 30개 미달 시 정직히 "찾은 만큼만 제출" — 빈 약속 금지

[최종 제출물]

- 카테고리 A~E 각 5~10개 항목 (총 30+ 항목)
- YAML 블록 30+ + 마크다운 요약 표
- 출처 URL 목록 (라이브 검증 통과 분만)
- 한계 명시 (찾지 못한 영역·학파 이견·검증 실패 출처)
