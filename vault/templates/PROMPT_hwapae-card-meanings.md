## 요구사항

### 1. 48패 결정론 카드 정의

화투 12달(月) × 4장 = 48패 각각의 객관 정의:

```json
{
  "month": 1,
  "name_ko": "송학(松鶴)",
  "name_japanese_origin": "마츠 (松)",
  "category": "광(光)" | "열끗(十)" | "띠(短)" | "피(皮)",
  "score_in_game": 20 | 10 | 5 | 1,
  "traditional_symbol": "소나무·학·일출",
  "traditional_meaning_summary": "장수·복록·길조",
  "season": "겨울 끝·봄 시작 (음력 정월)"
}
```

각 패에 대해:
- 객관 기호 (한자 명칭·일본 화찰 원형·계절·동식물)
- 게임 점수 (전통 화투 규칙)
- 전통 점패 의미 (객관 묘사만, 인과 예언 X)

### 2. 점패 풀이 결정론 규칙

화패 점패는 화투와 다른 결정론 풀이 패턴:
- 3장 뽑기 (과거·현재·미래 또는 사랑·일·재물)
- 카드 조합 의미 (예: 광 3장 = 큰 흐름, 띠 다수 = 작은 변화)
- 카드 순서 (1번 카드 = 핵심, 3번 카드 = 결과)

조사 항목:
- 한국 화투 점패의 전통 풀이 규칙 (배열법)
- 카드 조합·순서 의미 결정 규칙
- 결정론 알고리즘으로 변환 가능한 규칙만 채택

객관 출처 의무:
- 한국 전통 놀이·민속 사전 (국립민속박물관 한국민속대백과사전)
- 한국학중앙연구원 한국민족문화대백과사전
- KCI 등재 한국 민속학 논문

### 3. 절대 금지 (ADR-006·010 위반 영역)

다음은 카드 의미 정의에 절대 포함 금지:

- ❌ "이 카드는 큰 부자가 됩니다" 같은 단정적 예언
- ❌ "수명·건강·이혼" 인과 예언
- ❌ "이 조합은 흉합니다" 같은 절대 흉화
- ❌ 검증 불가 출처 ("○○ 무속인 말에 따르면")
- ❌ 가짜 인용 (저자·연도·서지 검증 가능해야)
- ❌ 일본 점패 직접 차용 (한국 화패 전통 우선)

본 시스템 페르소나(화선 낭자)는 "이 화패가 보여주는 건…" 같은 경향성
표현만 허용. 카드 객관 의미만 정의하고, 사용자 출력 표현은 LLM 페르소나가
담당.

### 4. 학파 회피 (ADR-002 정신)

화패 점패도 학파 차이 존재 가능:
- 전통 민속 학파
- 현대 운세 학파
- 일본 화찰 직접 영향 학파

본 SaaS는 학파 분쟁 회피 — **하나의 객관 출처로 통합 가능한 정의만 채택**.
학파별 미세 차이는 카드 메타데이터에 명시 (`school_variants` 필드), 본
시스템은 표준 학파만 디폴트 사용.

### 5. 검증 데이터셋 (의무)

각 점패 카드 조합 30건 + 객관 풀이 (사용자 노출 시 LLM이 페르소나 어조로
변환할 원본 데이터):

```json
{
  "spread": ["1월 광 (송학)", "5월 광 (난초)", "12월 광 (오동)"],
  "interpretation_facts": [
    "광 3장 동시 출현 — 큰 흐름",
    "송학·난초·오동 = 사군자 중 3가지 — 학문·기품 상징",
    "1월·5월·12월 = 사계절 안배 — 균형"
  ],
  "permitted_emphasis": "균형·기품·꾸준한 흐름",
  "forbidden_expressions": [
    "큰 부자",
    "이혼·단명",
    "확정적 예언",

  ],
```

본 30건은 본 시스템 회귀 테스트용.
