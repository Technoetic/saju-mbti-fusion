# 딥리서치 프롬프트 — 십이운성(장생법) 결정론 매핑

## 사용법

본 프롬프트는 본 시스템 사주 모듈(`engine/saju/`)의 결정론 보강용.
현재 신살 5종(천을귀인·문창귀인·역마·도화·공망)은 구현되어 있으나
**십이운성(장생·목욕·관대·건록·제왕·쇠·병·사·묘·절·태·양) 미구현**.

ADR-015 옵션 B 병행 패턴 적용 — 옵션 A(학파 회피 단순 카운팅) 디폴트
유지 + 십이운성을 **추가 정보**로 학파 명시 채택.

---

## 프롬프트 본문

```
사주 명리학 십이운성(十二運星) 장생법을 결정론 매핑 데이터로
조사·정리해주세요. 본 자료는 운세 SaaS 백엔드의 사주 풀이 추가 정보용이며,
ADR-002 학파 회피 + ADR-010 사실성 분리 + ADR-015 옵션 B 병행 패턴을
따라야 합니다.

### 요구사항

#### 1. 십이운성 결정론 매핑 테이블

일간(10천간) × 지지(12지지) = 120 조합의 12운성 매핑:

```yaml
twelve_stages_table:
  - day_stem: "甲"
    by_branch:
      子: "목욕"
      丑: "관대"
      寅: "건록"
      ...
    source: "<KCI URL: 자평진전 또는 적천수 인용 학술 논문>"
```

10천간 전체 매핑 필수 (甲乙丙丁戊己庚辛壬癸).

#### 2. 학파별 차이 명시

십이운성은 학파별 미세 차이 존재:
- 자평진전(子平眞詮) 청대 심효첨
- 적천수(滴天髓) 청대
- 연해자평(淵海子平) 명대 서승

본 시스템 채택 정책 (ADR-015 패턴):
- **단일 학파 명시 채택** (예: 자평진전)
- 학파 내 이견 명시
- 옵션 A(단순 신살) 디폴트 유지

#### 3. 학술 출처 라이브 검증

- KCI 등재 명리학 논문 (자평진전·적천수 학파 비교)
- 한국학중앙연구원 한국민족문화대백과사전
- ISBN 검증 가능 학술 서적

가짜 인용 절대 금지. 모든 출처 라이브 검증 가능해야 함.

#### 4. 본 시스템 결손 영역

현 `engine/saju/` 모듈 점검:
- ✅ pillars.py (만세력)
- ✅ ten_gods.py (십성)
- ✅ wuxing.py (오행)
- ✅ shensha.py (신살 5종 — 천을·문창·역마·도화·공망)
- ✅ name_saju_school.py (이재승 옵션 B 억부)
- ❌ 십이운성 미구현

신규 모듈 후보:
- `engine/saju/twelve_stages.py` 결정론 함수
- 회귀 30건 (학파 검증 데이터)

#### 5. 사용자 출력 면책 가이드

다음 표현 금지:
- ❌ "장생에 해당하니 평생 길합니다." (인과 단정)
- ❌ "사·묘에 해당하니 흉합니다." (예언)

채택할 표현:
- ✅ "자평진전 학파 기준 일간 甲, 월지 寅은 '건록(建祿)'에 해당합니다.
  명리학 학설이며 인과 길흉이 아닙니다."
- ✅ "본 결과는 자평진전 단일 학파 채택이며, 적천수·연해자평 등 다른
  학파에서는 미세 차이가 있을 수 있습니다."

#### 6. 회귀 테스트 데이터셋

본 시스템 회귀 검증용 30쌍 이상:

```json
[
  {
    "id": "stage_001",
    "day_stem": "甲",
    "month_branch": "寅",
    "expected_stage": "건록",
    "school": "자평진전",
    "source_url": "<KCI URL>"
  }
]
```

#### 7. 옵션 B 병행 패턴 적용 (ADR-015 정합)

ADR-015 옵션 A(디폴트) + 옵션 B(이재승 추가) 패턴 그대로:

| 항목 | 옵션 A (디폴트, ADR-002) | 옵션 B 후보 (본 프롬프트) |
|---|---|---|
| 신살 | 5종 결정론 (학파 무관) | 십이운성 (자평진전 학파 명시) |
| 사용자 노출 | 항상 | 선택적 (관심 사용자) |
| 면책 | "객관 매핑" | "<학파>의 학설" |

### 출력 형식

1. **십이운성 매핑 표 YAML** (120 조합 전체)
2. **학파별 차이 표** (자평진전 vs 적천수 vs 연해자평)
3. **학술 출처 표** (KCI URL + DOI 라이브 검증)
4. **본 시스템 모듈 명세** (twelve_stages.py 함수 시그니처)
5. **사용자 출력 면책 가이드라인**
6. **회귀 데이터셋 JSON** (30쌍)

### 검증 기준

- 모든 학술 인용 KCI/RISS 라이브 검증 가능
- 가짜 인용 0건
- 학파 명시 의무 (단일 학파 채택)
- 인과 길흉 표현 0건
- 회귀 30쌍 이상

위 조건 미충족 시 ADR-010 사실성 분리 + ADR-002 학파 회피에 따라
거부됩니다.
```

## 본 시스템 채택 절차

1. `/squeeze-report` 사실성 분리 + KCI 라이브 검증
2. ACCEPT 후보:
   - 신규 ADR (옵션 C — 십이운성 자평진전 학파 채택, ADR-015 패턴 반복)
   - `engine/saju/twelve_stages.py` 신규
   - 회귀 30건
   - vault/references/ 자평진전 출처 영속화
3. ADR-002 디폴트 유지 (단순 신살은 영향 0)

## 면책

- ADR-002 학파 회피는 디폴트 출력에 유지
- 옵션 C(십이운성) = 단일 학파 명시 (자평진전 또는 적천수 1개)
- 사용자 출력에 학파·면책 자동 포함
- 합충형해파 등 동적 변화는 본 옵션 X (의도된 제약)
