한국인 얼굴 부위별 형상 분류를 위한 학술 논문 본문 정량 통계 추출을 의뢰합니다.

[배경 컨텍스트]

얼굴 사진을 입력받아 한국 인상 분석을 제공하는 한국어 SaaS 제품에서,
부위별 형상을 결정론적으로 분류하는 모듈을 개발 중입니다.

1단계로 입꼬리 분류(올라간·내려간·수평)는 학술 출처 기반으로 본문화 완료.
2단계인 본 의뢰는 코·눈·턱 3부위 분류의 구체 임계값을 추출하기 위함입니다.

이전 1차 딥리서치 보고서가 학파 통설 명칭과 측정 지표는 제시했으나
임계값을 "평균 + 0.5*SD (예시)" 형태로만 표기하고 실제 평균(μ)·표준편차(σ)·
표본 크기(N) 수치를 명시하지 않아 결정론 분류기를 구현할 수 없습니다.
본 의뢰는 그 공백을 메우는 것이 목표입니다.

[조사 목표]

다음 4개 학술 출처의 원논문 본문(PDF·DBpia·KoreaScience·PMC 공식 페이지)을
직접 fetch하여 한국인 안면 인체측정학 정량 통계(μ·σ·N)를 추출하라.
각 항목은 원문 직접 인용 + 출처 페이지 번호 또는 표 번호 명시 필수.

[조사 대상 출처 4종]

출처 A. 송우철 외 (2017). 한국 성인의 얼굴 형태 인체측정학 분석.
- DBpia URL: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE07133895
- 추출 대상 통계:
  · 콧방울 너비 (nostril wing width) μ·σ·N (성별 분리 권장)
  · 콧방울 너비 / 얼굴 너비 비율 분포
  · 코 길이 (nose length) / 얼굴 세로 비율 분포
  · 코끝 둥금 정도(가능하면)

출처 B. 한국인 비강 형태 cone-beam CT 분석 (2024).
- PMC URL: https://pmc.ncbi.nlm.nih.gov/articles/PMC11431719/
- 추출 대상 통계:
  · 콧대 굽음 정도 (bridge curvature) 분포
  · 코끝 각도 (nasal tip angle) μ·σ
  · 매부리코 분류 기준이 논문 내 존재 시 그 임계값
  · 들창코 / 정상 코 분류 임계값 (콧구멍 노출각)

출처 C. 최윤경, 이경희 (2009). 한국 여성 안면 형태 분류 연구.
- DBpia URL: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE09916734
- 추출 대상 통계:
  · 눈 가로/세로 비율 (eye aspect ratio) μ·σ
  · 눈 너비 / 얼굴 너비 비율
  · 홍채 위·아래 흰자위 노출 비율 분포 (삼백안 임계값 도출 가능 시)
  · 눈매 외측 기울기 (inner-outer corner tilt) 각도 μ·σ

출처 D. 노상훈 외 (1998). 한국인 하악각 통계 분석.
- JKAOMS URL: https://www.jkaoms.org/journal/download_pdf.php?spage=129&volume=32&number=2
- 추출 대상 통계:
  · 하악각(jaw angle) 각도 μ·σ·N (성별 분리)
  · 사각턱 / 둥근턱 분류 임계값 존재 시 그 기준
  · 턱 너비 / 턱 길이 비율 분포 (가능 시)

[요구사항 — 각 출처별 반드시 답할 것]

각 출처 A~D에 대해 원논문 본문을 직접 fetch한 결과를 다음 형식으로 정리:

1. fetch 성공 여부 (HTTP 200 + 본문 접근 가능):
2. 논문 정확 서지 (저자·연도·학술지·권·호·페이지·DOI):
3. 표본 크기 N (전체 / 성별 분리):
4. 측정 항목별 통계:
   - 항목명 (한글·영문):
   - 평균 μ (단위 명시):
   - 표준편차 σ:
   - 측정 방법 (직접 측정·이미지·CT·MRI 등):
   - 본문 또는 표 번호:
5. 분류 임계값이 논문 본문에 명시된 경우 그 기준:
   - 예: "사각턱은 하악각 < 110도" 같은 명시 기준
6. 한계 명시 (표본 한정·연령대·측정 방법 차이 등):

위 4개 출처 외에 다음 영역에서 한국인 표본 학술 논문을 추가 발굴:

[출력 형식 — YAML 스켈레톤 + 마크다운 표]

각 출처별 YAML 블록:

```yaml
- source_id: A
  citation: "송우철 외 (2017). 한국 성인의 얼굴 형태 인체측정학 분석. 한국디자인포럼, X(Y), p.NN-NN."
  url: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE07133895
  fetch_status: success | partial (abstract only) | failed
  fetch_date: YYYY-MM-DD
  sample_size:
    total: N=...
    male: N=...
    female: N=...
  measurements:
    - item_ko: 콧방울 너비
      item_en: nostril wing width
      mean: 32.5
      sd: 3.1
      unit: mm
      method: 직접 측정 (Martin 방식)
      table_or_page: Table 2, p.5
    - item_ko: 콧방울 너비 / 얼굴 너비 비율
      ...
  threshold_in_paper:
    - rule: "복코 분류 기준 (있는 경우)"
      criteria: "콧방울 너비 비율 > mean + 0.5*SD = ..."
  caveats:
    - "표본 20-30대 한정"
```

마지막에 마크다운 요약 표:

| 출처 ID | fetch | 표본 N | 추출 통계 수 | 분류 임계값 명시 |
|---|---|---|---|---|

[검증 기준]

- 원논문 본문 직접 fetch 의무 — abstract 또는 인용 논문 요약만으로는 채택 불가
- fetch 실패 시 정직히 "abstract only" 또는 "failed" 표시
- 가짜 통계 금지 (μ·σ가 본문에 없으면 "본문에 명시 부재"로 정직 보고)
- 표본이 한국인 외(백인·중국인 등)면 명시 — 한국 SaaS 적용 부적합 표시
- 사상체질 비교 통계가 논문에 있어도 본 의뢰 출력에 포함 금지 (한의학 도메인은
  본 SaaS 채택 범위 외)
- 분류 임계값이 원논문에 부재할 경우 "임계값 부재 → mean ± 0.5*SD 권장값
  도출 가능" 표시

[최종 제출물]

- 출처 4종 × 추출 통계 평균 3-5개 = 12+ 정량 항목
- 마크다운 요약 표
- 한계 명시 (fetch 실패 출처·논문 본문 비공개·표본 한계)
- 추가 발굴 출처 목록 (KCI/DBpia/PMC 등)
- 사상체질 인용 부분은 본 출력에서 의도적으로 제외했음을 정직히 명시
