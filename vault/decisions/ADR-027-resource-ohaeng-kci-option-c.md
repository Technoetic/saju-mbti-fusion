# ADR-027 — 자원오행 옵션 C (KCI 학설 매핑) 94자 본문화

## 상태

Accepted (2026-05-17).

## 맥락

ADR-026 (2026-05-17)에서 대법원 인명용 한자 9,932자를 본문화했으나,
**자원오행 매핑은 부수 기반 자동 매핑(옵션 A) 35.5%(3,524자)에 한정**.

자원오행 매핑 방법에는 3가지 옵션이 있음:

| 옵션 | 매핑 기준 | 장점 | 단점 |
|---|---|---|---|
| A | 부수 (214) 자동 매핑 | 결정론·기계 처리 가능 | 자원·본의 불일치 |
| B | 한자 의미(자의) 직접 매핑 | 직관적 | 주관 개입·재현 불가 |
| C | **자원·본의 형태론 추적** | 학술 정합 | 공수 다대 |

본 시스템은 옵션 A 부수 자동 매핑만 적용했으나, 외부 보고서
"자원오행 5001자 매핑 조사"(2026-05-17)가 옵션 C 방법론과 94자
샘플 매핑 데이터를 KCI 학술 출처 기반으로 제공.

본 보고서 본문화 시 **충돌 6건 검출** (옵션 A 부수 ≠ 옵션 C KCI):

| 한자 | 부수 자동 (A) | KCI 학설 (C) | 사유 (KCI) |
|---|---|---|---|
| 仁 | 수 (부수 9 人) | 목 | 오상의 인(仁)=목 |
| 信 | 수 (부수 9 人) | 토 | 인언일치 = 중재의 토 |
| 誠 | 목 (부수 149 言) | 토 | 信과 동질 = 진실의 토 |
| 春 | 화 (부수 72 日) | 목 | 봄 = 생명의 목 |
| 暗 | 화 (부수 72 日) | 수 | 어둠 = 정적의 수 |
| 作 | 수 (부수 9 人) | 화 | 창조 = 동적의 화 |

## 결정

**옵션 A 디폴트 + 옵션 C 신규 필드 추가**. 옵션 C를 우선 적용하되
옵션 A는 보존 (학설 충돌 명시).

### 구현

`data/korean_hanja_unihan.json` 각 entry에 신규 3 필드 추가:

```json
{
  "char": "仁",
  "hangul": "인",
  "kangxi_strokes": 4,
  "radical": 9,
  "resource_ohaeng": "수",
  "resource_ohaeng_kci": "목",
  "kci_reason": "사람(亻)이 친밀하게 교류함 ... 오상의 목(木)",
  "kci_school_source": "김기승(2024), 이재승(2025)"
}
```

스키마 변경 0 (기존 필드 보존, 신규 필드 추가).

### API

`engine/divination/name_unihan.py`:

| 함수 | 반환 | 주체 |
|---|---|---|
| `resource_ohaeng(char)` | 부수 자동 (옵션 A) | 기존 |
| `resource_ohaeng_kci(char)` | KCI 학설 (옵션 C) | **신규** |
| `kci_reason(char)` | 매핑 근거 (자원·본의) | **신규** |
| `kci_school_source(char)` | 학파 출처 | **신규** |
| `preferred_ohaeng(char)` | KCI 우선, 없으면 부수 | **신규** |
| `total_with_kci()` | KCI 매핑 한자 수 (94) | **신규** |

### 데이터 범위

- ADR-027 본문화 시점: **94자** (보고서 §대규모 매핑 데이터)
- 구성: 오상 5자 + 계절 4자 + 방위 4자 + 추상부수(宀·亠·力·勹·又) + 빈출 회의/형성문자
- 매핑 분포: 9 동의(A=C) + 6 충돌 + 79 신규 채움(A 빈→C)

## 면책 (ADR-010 준수)

사용자 출력에 KCI 매핑 표시 시 다음 의무:

1. **학파 출처 동반 노출**: `김기승(2024), 이재승(2025)` 등
2. **부수 자동 매핑과 충돌 시 명시**: "본 한자는 부수 자동 매핑(수)과
   학설 자원오행(목)이 다릅니다. 본 시스템은 학술 우선 (옵션 C) 적용."
3. **예언적 인과 금지**: 자원오행이 사용자의 운세/길흉/건강과 인과
   관계를 단정하지 않음
4. **결정론 보장**: 동일 한자 → 동일 매핑 (lru_cache + JSON)

## 한계 (영구 기록)

- 본 ADR 94자는 보고서 §대규모 매핑 데이터에서 본문화된 분량.
  보고서가 표제로 명시한 "5001자 매핑 데이터"는 미충족 (94자 = 1.9%).
- 보고서 §6 disputed 5자(辰·康·詠·豕·智) primary+alternative 구조는
  본 ADR 범위 외 (스키마 변경 필요 → 별도 ADR-028 후보).
- 신규 1,407자(ADR-026)에 대한 KCI 매핑 0 (학술 출처 부재).
- 본 ADR은 **immutable**. 데이터 추가 시 새 ADR.
