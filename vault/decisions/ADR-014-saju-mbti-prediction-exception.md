# ADR-014: 사주 → MBTI 4축 경향성 추정 — 거부 정책의 예외

## 배경

사용자 결정 (2026-05-17): "이건 예외다."

본 프로젝트는 [[../reports/saju-mbti-correlation]] (사주-MBTI 매핑 보고서)를
ADR-010 사실성 분리 원칙으로 거부 사례 1호로 영속화. 거부 사유:
- 인용 학술 검증 불가
- 학파별 매핑 차이 큼 (ADR-002 학파 회피 위반 위험)
- "예측" 단정 = 페르소나 절대 규칙(단정적 예언 금지) 위반

사용자가 본 거부 결정에 대해 **명시적 예외 결정**을 내림. 본 ADR로
예외 영속화 + 위반 위험 통제 규칙 명시.

## 결정

사주 풀이 시 사주 → MBTI 4축 **경향성** 추정 + 표시 허용. 단:

### ✅ 허용

1. 4축 경향성 표시 (E/I, S/N, T/F, J/P 각각)
2. "결이 비치는구먼" 형식 경향성 표현
3. 사용자가 본인 MBTI 입력 → 비교 풀이
4. 본 시스템 자체 매핑 규칙임을 명시 (학설 권위 가장 X)

### ❌ 절대 금지

1. **16유형 단정** ("당신의 MBTI는 ENFJ입니다") — 영구 금지
2. 4축 모두 단정 ("E + N + F + J") — 단정 누적도 금지
3. 학설·연구 권위 인용 ("○○ 연구에 따르면")
4. 예측 정확도 표시 (자기실현적 예언 회피)
5. "MBTI 검사를 대체합니다" 같은 표현 (한국 심리검사법 회피)

## 매핑 규칙 (옵션 A 정신 유지)

ADR-002 단순 오행 카운팅 + 음양 비율 기반. **본 시스템 자체 규칙**:

| MBTI 축 | 본 시스템 매핑 | 미정 조건 |
|---|---|---|
| E vs I | 사주 양 비율 ≥ 0.55 → E, ≤ 0.45 → I | 0.45 < ratio < 0.55 |
| S vs N | (土+金)/total ≥ 0.4 → S, (木+水)/total ≥ 0.4 → N | 둘 다 < 0.4 |
| T vs F | 金 비율 + 양간(甲丙戊庚壬) → T, 火/水 비율 + 음간 → F | 명확 신호 부재 |
| J vs P | 오행 분포 표준편차 < 0.5 → J (안정), > 1.0 → P (편중) | 0.5 ≤ stdev ≤ 1.0 |

**미정**은 명시적 결과 — 단정 회피의 핵심 안전 장치.

## 출력 형식 의무

만월 아씨 페르소나 출력 시:

✅ 허용:
> "그대의 사주를 보니, 외향(E)의 결이 비치고, 직관(N)의 기운이 흐르는구먼.
> 사고(T)와 감정(F) 사이는 어느 한쪽으로 기울지 않았으니, 그대 본인이
> 알고 계신 MBTI와 견주어 보시게."

❌ 금지:
> "당신의 MBTI는 ENFJ입니다."
> "사주가 ENFJ를 예측합니다."

## 면책 의무

사용자 출력 시 다음 면책 의무 (페르소나 또는 footer):

> "사주에서 보이는 결을 MBTI 4축으로 견주어 본 것이며, 본 시스템의 자체
> 매핑일 뿐 정통 명리학·심리학 진단이 아닙니다. MBTI 검사를 대체하지
> 않습니다."

## 검토한 옵션

### A. 거부 유지 (이전 결정)
- 장점: ADR-002·010 정신 보존
- 단점: UX 가치 (사용자 MBTI 부담 ↓) 포기

### B. 본 시스템 자체 매핑 + 경향성 표현 (채택)
- 장점: UX 가치 + 단정 회피 + 학파 분쟁 회피
- 단점: ADR-002·010 부분 예외 — 본 ADR로 영속화 필요

### C. 단정적 예측
- 장점: 가장 단순
- 단점: 모든 페르소나 규칙 위반 + ADR-010 위반 + 법적 위험

## 채택

**B 채택**. 사용자 예외 결정 (2026-05-17) 영속화.

### 본 ADR이 ADR-002·010 정신에서 다른 점

- ADR-002 (학파 회피): 본 ADR은 학설 인용 0 — 본 시스템 자체 매핑이라
  학파 분쟁 무관. 옵션 A 정신 그대로
- ADR-010 (사실성 분리): 본 ADR은 학설 권위 가장 X. "본 시스템 매핑" 명시 +
  미정 조건 명시 = 사실성 분리 정신 보존

→ **ADR-002·010을 superseded 하지 않음**. 부분 예외.
