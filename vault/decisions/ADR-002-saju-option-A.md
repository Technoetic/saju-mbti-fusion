# ADR-002: 사주 용신은 옵션 A (단순 오행 분포) 채택

## 배경

보고서 §3은 만세력 + 용신 도출을 작명 추천에 활용하라고 강조.
그러나 용신 도출은 학파별로 다름:
- 격국론(자평진전): 격국 + 십신 균형
- 조후론(궁통보감): 한습조열 보정
- 통관론: 상극 사이 매개 오행
- 억부론(적천수): 일간 강약 부조·억제

같은 사주를 4 학파에 보내면 4 다른 답. 책임 소재 모호.

## 검토한 옵션

### A. 단순 오행 분포만 노출 (채택)
- 사주 8글자 → 오행 분포 (천간 1점 + 지지 지장간 가중치)
- 가장 적은 오행 = 추천 자원오행
- 학파 무관, 객관적

### B. 단일 학파 채택 (조후/억부 중 1개)
- 결정론 알고리즘 가능, 한국 시장 친화
- 다른 학파 신봉자 불만 + 명리학자 검수 필요

### C. 사용자가 직접 target_ohaeng 입력
- 가장 안전, UX 최악

## 채택

**A 채택**. 이유:
- 학파 분쟁 회피
- 결정론
- 운영표준 §7.1 면책(법률 자문 거부)과 일관
- 추가 사용자 데이터 입력 부담 X

## 결과

- `engine/divination/name_saju_ohaeng.py` 신규
- `engine/saju/pillars + wuxing` 기존 자산 재사용
- `name_gaeja/gaemyeong`에 target_ohaeng 직접 주입 가능
- 사용자 응답에 "참고용·학파별 차이 있음" 면책 명시

## 면책

- 본 옵션은 명리학적 "용신 판정"이 아님 — 단순 통계
- "당신의 사주에 木이 가장 적어 木 자원오행 추천" 식의 균형 보강 지표
- 정통 용신 도출은 향후 옵션 B(명리학자 검수 후)에서 별도 추가 가능

## 향후 전환 가능성

옵션 B(단일 학파 채택)는 다음 조건 만족 시 검토:
- 학파 채택 의사결정 ADR 작성
- 딥리서치 보고서 받음
- 면책 강화
- 옵션 A와 병기 (사용자 선택 가능)
