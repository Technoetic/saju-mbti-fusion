# ADR-015: 사주 용신 옵션 B — 이재승 계량화 억부론 채택

## 배경

[[../reports/saju-option-B-school]] (🟢 외부 입력 대기)에 대한 딥리서치
보고서 수신 (`사주 용신 학파 채택 연구 보고서.md`). ADR-010 사실성 분리
검증 결과 **본 프로젝트 모든 외부 보고서 중 가장 정합도 높음**:

- KCI 등재 논문 출처 라이브 검증 완료 (DBpia/koreascholar)
- 가중치·임계치가 논문 원문과 일치
- 학파 내 이견 명시 (의도된 설계 제약)
- ADR-010 면책 의무 자체 인용

## 결정

본 시스템에 옵션 B 단일 학파(이재승 계량화 억부론)를 **추가 정보**로 채택.
ADR-002 옵션 A(단순 오행 카운팅)는 **디폴트 유지**.

### 옵션 A vs 옵션 B 역할 구분

| 항목 | 옵션 A (ADR-002, 디폴트) | 옵션 B (ADR-015, 추가) |
|---|---|---|
| 출력 | 추천 자원오행 (작명) | 용신 (명리학) |
| 근거 | 단순 오행 카운팅 | 이재승 계량화 (KCI) |
| 학파 회피 | ✅ 학파 무관 | ❌ 억부론 채택 |
| 사용자 노출 | 항상 | 선택적 (관심 사용자) |
| 면책 | "균형 보강 지표" | "이재승 계량화 억부론 학설" |

본 결정은 ADR-002를 **superseded 하지 않음**. 두 옵션 병행.

## 검토한 옵션

### A. 옵션 A 그대로 + 옵션 B 추가 (채택)
- 장점: ADR-002 학파 회피 정신 보존 + ADR-014도 옵션 A 위에 구축됨 영향 0
- 장점: 사용자에게 단일 학파 정보 추가 제공
- 단점: 두 결과 병행 노출 시 사용자 혼란 가능 — 면책 의무로 통제

### B. ADR-002 superseded
- 장점: 일관성 1개 학파
- 단점: ADR-002·014 정신 변경 → 연쇄 영향
- 단점: 학파 회피 정책 포기

### C. 영속화만, 본문화 보류
- 장점: 안전
- 단점: 합리적 본문화 가능한데 보류 = 비합리

## 채택

**A 채택**. 옵션 A 디폴트 + 옵션 B 추가.

- "이재승 교수의 계량화 억부론 학설을 채택한 참고용 정보"
- "합충형해파 등 명리학 동적 변화는 반영하지 않음"

## 면책

- 본 ADR은 ADR-002를 **superseded 하지 않음** (병행)
- 옵션 B는 단일 학파 채택 — 다른 명리학 유파 결과와 다를 수 있음
- 합충형해파·외격·종격 등은 의도된 설계 제약 (학술적 정직성)
- 사용자 출력에 학설 명시 + 학파 한계 면책 의무
