# 사주명리학-MBTI 연관성 — 사실성 검토 (높은 위험)

## 보고서 요약

사주 음양 ↔ MBTI E/I, 오행 ↔ 기질, 십성 ↔ S/N/T/F 기능을 매핑하고 "안수현(2023)"·"최영(2023)"·"백강희·정미숙(2022/2024)" 등 학술 연구를 인용하여 통계적 유의성 주장.

## 🟢 팩트

| 주장 | 검증 |
|---|---|
| MBTI의 4 양극성 + 16 유형 구조 | ✅ 사실 |
| 융 8 인지 기능 스택 | ✅ 본 프로젝트 [engine/saju/mbti_functions.py](../../engine/saju/mbti_functions.py)에 이미 표준 매핑 |
| 음양오행·십성·대운/세운 명리학 개념 실존 | ✅ 명리학 고전 |
| 자평명리학·자기보고식 검사 차이 | ✅ 인식론적 사실 |

## 🟡 구조

- "사주 = 선천 원형 / MBTI = 현재 스냅샷" 추상화 — 의미 있는 비유
- 사주 + MBTI 결합 활용은 MZ세대 트렌드로 실제 존재 (서비스 다수)

## 🔴 도그마 (사용 금지)

### 1. ❌ 인용된 학술 연구의 실재성 미검증

- "안수현 (2023) — 명리 상담자 50명 표본 음양 비율 통계"
- "최영 (2023) — 일원배치분산분석 오행-MBTI"
- "백강희·정미숙 (2022, 2024) — 십성 vs S/N/T/F"

**판정**: 검증 불가능. AI 생성 보고서의 **가짜 인용(citation hallucination)** 가능성 매우 높음. 본 보고서의 "통계적 유의성" 주장 전부 의심.

### 2. ❌ "구조적 상동성", "통계적으로 매우 유의미"

- 표본 크기·p-value·재현성·연구 설계 0건 명시
- "16 MBTI 유형과 오행의 지배적 특성 간에 매우 일관된 상호 배속 패턴" — 검증 불가

### 3. ❌ MBTI 자체의 재현성 논쟁 미언급

- MBTI는 학계에서 재현성·구성 타당도 논쟁 큼 (NEO-PI-R / Big5와 대비)
- 본 보고서는 MBTI를 **검증된 도구**로 전제 — 부정확

### 4. ❌ 사주 십성 → MBTI 매핑 (식상=NF / 재성=ST / 관성=SJ / 인성=NT/I)

- 명리학 전통 학파별로 십성 해석 차이 큼 ([[../decisions/ADR-002-saju-option-A]] 학파 회피 정신과 충돌)
- MBTI 매핑은 본 보고서 주장이며 명리학 통설 아님

### 5. ❌ "운명론적 통계학" 같은 권위 수사

- 명리학을 "통계학"으로 칭하나 현대 통계학 정의와 무관 (n=?, 표본 추출 방법=?)

## 본 프로젝트 영향 평가

### 위험 평가

본 보고서를 본 프로젝트에 반영할 경우 발생 가능한 위험:

1. **CLAUDE.md §0 위반**: "결정론 엔진 + LLM 작문 분리, **학파 분쟁 회피**" — 본 보고서는 특정 매핑 학설을 채택하라고 요구
2. **ADR-002 위반**: 사주 옵션 A(학파 분쟁 회피, 단순 오행 카운팅) → 보고서는 십성 학설 채택 필요
3. **ADR-010 위반**: 인용된 학술 연구 실재성 검증 불가 (가짜 출처 가능성)
4. **법적·윤리적 위험**: MBTI 결과를 사주로 "예측" 시 심리 검사법·면허 문제 가능

### 본 프로젝트 현 상태

본 프로젝트는 MBTI를 다음과 같이 사용 중:
- [engine/saju/mbti_functions.py](../../engine/saju/mbti_functions.py) — 16 유형 → 8 융 인지 기능 (표준 매핑, 학설 충돌 없음)
- [engine/saju/compat.py](../../engine/saju/compat.py) — 사주-MBTI 호환성 (?)
- [engine/saju/mbti_functions.py](../../engine/saju/mbti_functions.py) — 인지 기능 스택만

→ **현 코드는 융 표준 매핑만 사용**. 본 보고서가 주장하는 "십성-MBTI 매핑"은 미적용.

## 본 시스템 반영

### ❌ 채택 불가

본 보고서 핵심 주장(사주 음양/오행/십성 ↔ MBTI 매핑) **본 프로젝트 반영 금지**:

1. 인용 출처 검증 불가
2. 학파 분쟁 회피 원칙 위반
3. ADR-010 사실성 분리 — 도그마로 분류

### 🟡 시사점 (개념 차용 가능)

- "사주 = 선천 원형 / MBTI = 현재 상태" 추상화는 **사용자 인터페이스 설명 수준에서** 가능
  - 단, "통계적으로 입증됨" 표현 금지
  - "전통 관념 + 현대 자기 보고식 도구의 상보적 활용" 같은 객관 묘사로
- **출력 면책 필수**: "본 결과는 전통 관념과 자기 보고식 도구의 결합 해석이며, 학술적·심리 측정학적 진단이 아닙니다"

### ❌ 본 프로젝트 미적용 결정

- 사주-MBTI 통계 매핑 ML 모델 (별도 보고서 [[saju-mbti-ml-model]] 존재) — 같은 사유로 채택 불가
